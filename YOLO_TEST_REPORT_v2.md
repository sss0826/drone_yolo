# YOLO 模型测试报告

## 1. 模型信息

| 项目 | 详情 |
|------|------|
| 架构 | YOLOv8n (nano) |
| 参数量 | 3.0M parameters, 8.2 GFLOPs |
| 训练数据 | water_trash（水面垃圾数据集） |
| 训练/验证 | 2,230 张 / 638 张 |
| Epochs | 50 |
| 图像尺寸 | 640×640 |
| 优化器 | AdamW (lr=0.0011) |
| 性能 | mAP@50 = **0.860**, mAP@50-95 = **0.504** |

## 2. 类别定义

共 5 类水面垃圾：

| 类别 | 训练样本数 | mAP@50 | mAP@50-95 | 评价 |
|------|-----------|--------|-----------|------|
| **plastic** | 374 | 0.916 | 0.690 | 🥇 最强，样本充足 |
| **can** | 134 | 0.908 | 0.579 | 🥈 很好，特征明显 |
| **bottle** | 911 | 0.860 | 0.493 | 🥉 中等，形态变化大 |
| **carton** | 216 | 0.835 | 0.451 | ⚠️ 一般，与水面颜色接近时难识别 |
| **paper** | **74** | 0.781 | **0.310** | 🚨 最差，样本严重不足 |

## 3. 真实场景测试（网络图片）

使用网络公开的水面垃圾图片进行离线测试：

| 图片 | 场景描述 | 视角 | 检出数 | 漏检原因分析 |
|------|----------|------|--------|-------------|
| 0947.jpg | 湖面漂浮垃圾 | 平视 | 2 (can) | 远景小目标漏检 |
| 4636.jpg | 俯拍水面漂浮物 | 俯拍 | 4 (plastic/can) | 密集场景 NMS 抑制 |
| 7120.jpg | 河边堆积垃圾 | 平视近景 | 0 | 与训练数据视角差异过大 |
| 9378.jpg | 岸边泥地垃圾 | 岸边 | 0 | 背景复杂，非水面场景 |
| heliu... | 水面大面积漂浮 | 俯拍 | 3 (plastic) | 效果较好，检出率 60%+ |

### 测试结论

1. **俯拍水面漂浮场景**下表现良好，检出率 60~80%
2. **平视/岸边/堆积场景**漏检严重，与训练数据视角不匹配
3. **密集重叠目标**容易被 NMS 合并，导致漏检
4. **类别混淆**：蓝色圆柱形塑料瓶易被误判为 `can`（易拉罐）

## 4. 阈值调优

| 置信度阈值 (CONF) | 检出率 | 误报率 | 推荐度 | 说明 |
|-------------------|--------|--------|--------|------|
| 0.4 | 高 | 高 | ❌ | 误报太多，水面波纹/树叶易误检 |
| **0.5** | **中** | **中** | **✅ 推荐** | **平衡检出与误报，最终采用** |
| 0.6 | 中 | 低 | ⚠️ | 保守，小目标易漏检 |
| 0.7 | 低 | 低 | ❌ | 漏报严重，不建议 |

| NMS IoU 阈值 | 效果 | 说明 |
|--------------|------|------|
| 0.7 (默认) | 密集场景漏检多 | 重叠框被大量抑制 |
| **0.3** | **保留更多重叠框** | **最终采用，适合密集漂浮垃圾** |

## 5. 已知问题与风险

| 问题 | 影响 | 缓解措施 |
|------|------|----------|
| **paper 样本不足** | 纸类垃圾几乎无法检出 | 比赛前补充 paper 类样本（建议 200+ 张） |
| **视角敏感** | 非俯拍场景漏检严重 | 比赛时保持摄像头**垂直向下**，确保俯拍 |
| **小目标漏检** | 远景/小尺寸垃圾检出率低 | 8 米高度下目标较大，问题不严重 |
| **类别混淆** | bottle/can 误判 | 增加两类区分性样本（颜色、形状差异） |
| **mAP@50-95 偏低** | 定位框不够精准 | 延长训练到 100 epochs，或启用 mixup 增强 |

## 6. 比赛建议（8 米飞行高度）

- **飞行高度**：**8m 左右**，低空飞行确保垃圾目标在画面中占比较大（>100×100 像素），有利于 YOLOv8n 识别
- **摄像头角度**：尽量**垂直向下**（俯拍 90°），减少斜拍造成的形变。如果云台无法垂直，**60°~90°** 也可接受，但避免低于 45°（水面反光严重且垃圾形变）
- **飞行速度**：建议 **3~5m/s**，8 米高度下画面移动快，速度太快会导致运动模糊
- **航线规划**：8 米高度单帧覆盖面积约 10×7m（假设 90° FOV），需规划**重叠率 30%** 的来回航线，避免漏扫
- **光照**：避免正午强光（水面反光严重），建议上午 9~11 点或下午 3~5 点
- **水质**：浑浊水面垃圾对比度低，清澈水面效果更好
- **备用方案**：如果现场检测效果不佳，可临时调低 CONF 到 0.4 或启用 TTA（Test Time Augmentation）
